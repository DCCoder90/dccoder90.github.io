## Overview

"Discover Brewton" is a cross-platform mobile application designed to provide an insight into the small town of Brewton, Alabama. This project, built on the **Xamarin.Forms** platform, allows for a single C# codebase to create native applications for both iOS and Android.

The application is structured using a standard Master-Detail navigation layout, allowing users to switch between a main "Browse" page and an "About" page. The core "Browse" section is set up to display a list of items, each with a title and description.
